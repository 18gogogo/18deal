# 18deal
18Deal 自助放盤平台是專為幫助了買家、租客物色心水物業設計的平台，旨在地產市場上的資訊，增加市場的透明度，致力幫助地產行業能持續健康發展。

18Deal is a self-service property listing platform designed to help buyers and tenants find their ideal properties. It aims to increase market transparency and contribute to the sustainable and healthy development of the real estate industry.
